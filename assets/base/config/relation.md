# Relation Types

Configurable relation vocabulary for typed body links (`#@label` fragment syntax).
Loader: `tools/lib/references/relation-types.js`.

| label    | description                                 | reverse    |
| -------- | ------------------------------------------- | ---------- |
| source   | primary source / citation this doc draws on | sourced-by |
| related  | lateral "see also" relation                 | related    |
| based-on | this doc derives from / extends the target  | basis-for  |
